City Jump

  • Originality: What is original about your game? My game is original in that the graphics are specialliy designed for this game. The character is original as well as the object design. All of the designs were made using paint.net.
  • Technical Merit: What is interesting about your game technology? The game has random velocities for the seven drone enemies on the map. This adds a layer of randomness into the game since a slow moving drone could become the fastest speed quickly. There are also animations for the player colliding with the enemies that disapear when they land back on the starting pad.
  • Prototype Postmortem: What did you learn from this prototype? I learned how to make a basic game from the phaser tutorial.https://phaser.io/tutorials/making-your-first-phaser-3-game/part1 . I learned the basics of the game loop and how to design enemy objects as well as collision. I also learned how to get keyboard input and create sprite animations for the character. What was the easiest or hardest part of making it? The hardest part was getting it to work on github. The version on github doesn't reflect the localhost version since I was unable to figure out how to fix the player not dropping below the platform in a way that wouldn't effect the local version. Some other related difficulties were working with javascript and github as well as figuring out how to use the phaser methods.
