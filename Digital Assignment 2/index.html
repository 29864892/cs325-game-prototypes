Space Fighter

  • Originality: All of the art was created by myself through paint.net.
  • Technical Merit: There are two types of enemies with different movement patterns. The debris can move in the y direction while the enemies only move forward. There are also projectiles that collide with both the enemies and the debris which scores 100 and 50 points respectively. Each time an enemy is hit, all enemies gain a permanent speed boost as well to raise the difficulty.
  • Prototype Postmortem: What did you learn from this prototype? I learned how to make groups of enemies as well as projectiles. I also learned how to lower the volume of sounds and how to add a tutorial at the start of the game. I also learned how toimplement timers to make the code function better. What was the easiest or hardest part of making it? The hardest part was the projectiles because there is a bug in the game where shooting a missle the moment the enemy is hit creates a motionless object that stays on the screen. I was unable to figure out how to fix it so I simply removed the projectile limit which may cause the program to crash if it goes on for too long.
